# WooCommerce 折扣工具

一款強大且靈活的 WooCommerce 折扣管理外掛，讓您可以使用條件、優先級及多種折扣類型來建立複雜的折扣規則。

## 🎯 功能特色

### 核心功能
- **多種折扣類型**：百分比折扣、固定金額折扣、價格覆寫、買一送一（BOGO）、免運費
- **彈性條件設定**：依商品、分類、使用者角色、數量、購物車總額、日期區間設定條件
- **優先級系統**：透過優先級控制哪些折扣優先套用
- **可疊加折扣**：允許多個折扣疊加，或僅套用最優折扣
- **使用次數限制**：為每個規則設定最大使用次數
- **日期排程**：根據日期區間自動啟用折扣

### 管理後台功能
- **直覺的規則編輯器**：分頁式介面，包含「一般」、「條件」、「折扣」、「顯示」及「預覽」分頁
- **條件建構器**：視覺化條件建構器，支援 AND/OR 邏輯
- **批量操作**：一次啟用/停用、刪除多個規則
- **快速操作**：使用 AJAX 快速切換規則狀態
- **搜尋與篩選**：強大的篩選功能，快速找到所需規則
- **設定管理**：匯入/匯出設定、還原預設值

### 前台功能
- **商品頁面顯示**：以三種樣式顯示折扣表格（表格、徽章、文字）
- **購物車顯示**：在購物車中顯示已套用的折扣
- **結帳頁面顯示**：在結帳頁面顯示總節省金額
- **響應式設計**：在各種裝置上皆完美呈現
- **自訂訊息**：自訂折扣訊息與樣式

### 效能與安全性
- **快取系統**：內建快取機制，確保最佳效能
- **SQL 注入防護**：所有查詢皆使用預處理語句
- **XSS 防護**：所有輸出皆已正確轉義
- **高效評估**：具備短路邏輯的智慧條件評估

## 📋 系統需求

- **WordPress**：5.8 或以上版本
- **WooCommerce**：5.0 或以上版本
- **PHP**：7.4 或以上版本
- **MySQL**：5.6 或以上版本

## 🚀 安裝方式

### 從 WordPress 後台安裝
1. 下載外掛 ZIP 檔案
2. 進入 WordPress 後台的 **外掛 > 安裝外掛**
3. 點擊 **上傳外掛** 並選擇 ZIP 檔案
4. 點擊 **立即安裝**
5. 啟用外掛

### 手動安裝
1. 將 `discount-tools` 資料夾上傳至 `/wp-content/plugins/`
2. 透過 WordPress 後台的 **外掛** 選單啟用外掛
3. 前往 **WooCommerce > 折扣工具** 進行設定

## 📖 快速入門指南

### 建立您的第一個折扣
1. 前往 **WooCommerce > 折扣工具 > 新增規則**
2. **「一般」分頁**：輸入規則名稱、類型、狀態、優先級
3. **「條件」分頁**：新增條件（商品、分類、使用者等）
4. **「折扣」分頁**：選擇折扣類型與數值
5. **「顯示」分頁**：設定顯示選項
6. 點擊 **發布**

如需詳細說明文件，請參閱[使用指南](docs/USER-GUIDE.md)。

## 🔧 開發

### 環境設定
```bash
git clone https://github.com/famiya/discount-tools.git
cd discount-tools
composer install --dev
npm install
```

### 程式碼規範
```bash
composer run-script phpcs    # 檢查 PHP 程式碼
composer run-script phpcbf   # 修正 PHP 程式碼
npm run lint:js              # 檢查 JS 程式碼
```

### 測試
```bash
composer run-script test     # 單元測試
```
